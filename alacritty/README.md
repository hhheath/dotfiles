# Alacritty

I've been using alacritty for a long time. I honestly don't know why I chose it. I think maybe i was looking for a new terminal emulator to switch to after using iterm2 for a long time and wanting something a bit more customizable, open source, and fast. I also went through this phase where I was really really into the rust programming language, so I was trying to support and use as many rust tools as I could find. 

Alacritty solved all of those things. But, after looking into the switch from yaml to toml for configuration, and the only reason being "because we wanted to" is kinda lame. But, to be fair, I don't have a very extensive config -- really just changing font and some other small quality of life things (for me). 

Actually, now that i edit it down, i literally only change the font and the launch window dimensions.
