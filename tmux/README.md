# Tmux

I barely scratch the surface on what tmux is capable of, and i mean *thoroughly* underutilizing the program. 

I think the main things to note in the config are: 

- i bind the prefix to ctrl + space. i find this to be a better set up than backtick (which i used for about 2 years) because there are too many occasions where i need to use a backtick in my writing (codeblocks in markdown, template strings in javascript, etc.)
- i also bind the vertical and horizontal splits to `-` and `\` respectively. 
- i set the window interval to start at 1. this is just makes sense in my head. I know programatically the list starts at the 0th index, but i like things starting at 1.

I love tmux. One of my 2024 goals is to learn *more* features this year.
